## üöÄ Quick Start Options

### 1: GitHub Codespaces 
**Steps to get started:**

1. **Accept the Assignment in Canvas** 
   - You will need a GitHub account. It is recommended to activate the [GitHub Student Pack](https://education.github.com/pack) for additional benefits, though this is not required for completing the assignments.
   - Find the GitHub assignment link in Canvas and click it to accept the assignment. You can also find the repository in your GitHub account.
   - This creates your own copy of the repository with owner access. Teachers will also have access to your repository.
   - You will complete all tasks within this repository.

2. **Create a Codespace**
   - Navigate to your newly created repository in GitHub (the link is provided when you accept the assignment).
   - Click the green **"<> Code"** button
   - Select the **"Codespaces"** tab
   - Click **"Create codespace on main"**

3. **Start coding!**
   - Wait 30-60 seconds for your environment to load
   - Your browser-based VS Code will open automatically
   - All Python tools and extensions are pre-installed
   - You can start working on the tasks immediately.

> **üí° Tip:** Once your codespace is created, you can resume work by simply opening the same codespace. You do not need to create a new one each time. If you don't use your codespace for approximately two consecutive weeks, you'll receive an email notification that it will be deleted. To prevent this, simply open and use your codespace periodically. If it does get deleted, you can create a new one without losing work, since you'll be regularly committing and pushing your code. However, we strongly recommend working on a regular basis to stay on track with the course.
